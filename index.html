INTRODUCTION

DRIM GRID UI lets you divide your content horizonatlly into parts, which helps organize the content more eficiently and visually.
DRIM GRID UI by default uses 12 column system, which means you can divide your content into up to 12 parts. You can choose to divide into equal or unequal parts.

Click on "</>" icon to see the HTML code

For example:

COLUMN 1
COLUMN 2 
                
    <div class="grid">
        <div class="column six"> COLUMN 1 </div>
        <div class="column six"> COLUMN 2 </div>
    </div>

Inside the grid (".grid") element there are 2 "div" elements with class="column six" where
- "column" means it is a grid column/item
- "six" is the number of parts needed.

Another exmple: 

                
    <div class="grid">
        <div class="column four"> COLUMN 1 </div>
        <div class="column four"> COLUMN 2 </div>
        <div class="column four"> COLUMN 3 </div>
    </div>

3 equal parts => 4 + 4 + 4 = 12.

X + Y != 12

The columns' widths do not have to sum to up 12, they can be more or less.
- if the sum of column widths is less than 12, the corresponding number of "missing" parts will be left empty.
- if the sum of column widths is greater than 12, the columns will continue "from a new line", will be displayed below.

</>
